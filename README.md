## 🚀 Como Executar

### Pré-requisitos
- Node.js 18+
- Yarn ou npm

### Instalação

```bash
# Clone o repositório
git clone https://github.com/edsonjuniornarvaes/portifolio.git

# Entre na pasta
cd portifolio

# Instale as dependências
yarn install

# Execute o projeto
yarn dev
```

Acesse [http://localhost:3000](http://localhost:3000) 🎉
